Block header types (in the order that I discovered them):

0x82 (Type A) - 1 byte unknown, 2 byte data size,                   1 byte name length, name string
0x42 (Type B) - 1 byte unknown, 1 byte data size,                   1 byte name length, name string
0x81 (Type C) - 1 byte unknown, 2 byte data size,                   1 byte name length, name string
0xC2 (Type D) - 1 byte unknown, 4 byte data size,                   1 byte name length, name string
0xC1 (Type E) - 1 byte unknown, 4 byte data size,                   1 byte name length, name string
0x45 (Type F) - 1 byte unknown, 1 byte data size,                   1 byte name length, name string
0x4B (Type G) - 1 byte unknown, 1 byte data size,                   1 byte name length, name string
0x4A (Type H) - 1 byte unknown, 1 byte data size,                   1 byte name length, name string
0x61 (Type I) - 1 byte unknown, 1 byte data size, 2 byte unknown,   1 byte name length, name string
0x41 (Type J) - 1 byte unknown, 1 byte data size,                   1 byte name length, name string
0xA1 (Type K) - 1 byte unknown, 2 byte data size, 2 byte unknown,   1 byte name length, name string
0x47 (Type L) - 1 byte unknown, 1 byte data size,                   1 byte name length, name string
0x48 (Type M) - 1 byte unknown, 1 byte data size,                   1 byte name length, name string
0x46 (Type N) - 1 byte unknown, 1 byte data size,                   1 byte name length, name string
0x49 (Type O) - 1 byte unknown, 1 byte data size,                   1 byte name length, name string
0xE1 (Type P) - 1 byte unknown, 4 byte data size, 2 byte unknown,   1 byte name length, name string
0x6C (Type Q) - 1 byte unknown, 1 byte data size, 2 byte unknown,   1 byte name length, name string
